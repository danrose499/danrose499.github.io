# Web Portfolio

This is a simple web portfolio project that showcases my work and skills. 

## Project Structure

```
web-portfolio
├── src
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── scripts.js
│   └── index.html
├── assets
│   ├── fonts
│   └── images
├── .gitignore
└── README.md
```

## Getting Started

To set up this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd web-portfolio
   ```

3. Open the `index.html` file in your web browser to view the portfolio.
